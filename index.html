Virtual Assistant for the European Data Portal (Demo)

This prototype is designed to help you interact with the European Data Portal via a virtual assistant.

To start, please click on the chat icon in the bottom right corner and type "hi"

When searching for datasets, try asking something like:

"Find datasets about Estonia"

⚠️ Note: If the search does not load the first time, try to search again.
